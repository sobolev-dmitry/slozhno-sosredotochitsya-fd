# Проектная работа «Сложно сосредоточиться»


Адаптивный сайт с поддержкой светлой и тёмной тем, автоматически переключающихся в зависимости от системных настроек пользователя. Проект включает шапку, основной контент с текстовыми карточками, галерею изображений и подвал.

## 📸 Скриншот проекта


![Интерфейс сайта «Сложно сосредоточиться»](/images/screen-2.png)

*Главная страница: шапка с навигацией и кнопкой переключения тем, основной контент с карточками, галерея изображений и подвал сайта в тёмной и светлой темах.*

## 🎯 Цель проекта

Отработать навыки:
* адаптивной вёрстки по принципу Mobile First;
* работы с CSS‑переменными для реализации тёмной и светлой тем;
* автоматического переключения тем через `prefers‑color‑scheme`;
* принудительного переключения тем через интерфейс (кнопки);
* использования гридов и логических свойств CSS.


## ✅ Реализованные функции

- **Адаптивность** под разные разрешения (мобильные, планшеты, десктопы) с плавным масштабированием элементов.
- **Две цветовые темы** (тёмная и светлая) с переключением:
  * автоматически — по настройкам ОС (`prefers-color-scheme`);
  * вручную — через кнопки интерфейса.
- **Фиксированный фоновый рисунок** для всей страницы (`background-attachment: fixed`).
- **Шапка и подвал** с одинаковой структурой, декоративными элементами (уголки, надпись REC) и адаптивным заголовком (`clamp()`).
- **Основной контент** из нескольких секций с текстовыми карточками и стилизованными ссылками (эффекты при наведении и фокусе).
- **Галерея изображений** с «ленивой» загрузкой (`loading="lazy"`) и адаптацией через CSS Grid.
- **Кнопки переключения тем** с состояниями `:hover`, `:focus-visible` и активным классом.

## 🛠️ Использованные технологии

- **HTML5** — семантическая разметка, атрибуты доступности (`aria-*`), ленивая загрузка изображений.
- **CSS3**:
  * CSS Grid и Flexbox для раскладки;
  * логические свойства (`margin-block-start`, `border-inline-end`) для интернационализации;
  * CSS‑переменные (`--accent-color` и др.) в отдельных файлах (`variables.css`, `dark.css`, `light.css`);
  * медиазапросы для адаптивности (768px, 1024px);
  * `clamp()` для адаптивного размера текста;
  * псевдоэлементы для декоративных уголков;
  * `text-shadow` для эффектов текста.
- **JavaScript** — скрипт переключения тем (добавляет классы `.theme-light`, `.theme-dark`, `.theme-auto` к `<html>`).

## 📝 Примечания

- Вёрстка выполнена по макету с соблюдением Pixel Perfect.
- Стили организованы в отдельных CSS‑файлах: переменные (`variables.css`), базовые стили (`style.css`), темы (`dark.css`, `light.css`).
- Подключение фавиконок (`.ico`, `.svg`, `apple-touch-icon`).
- Метатег `<meta name="color-scheme" content="dark light">` для поддержки системных тем.
- Все декоративные элементы реализованы без лишних HTML‑тегов (псевдоэлементы, `aria-hidden`).
- Проверена корректность отображения в 6 сценариях (светлая/тёмная/авто-тема при разных настройках системы).
